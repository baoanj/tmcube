# tmcube

> 基于现代Web技术的高校教学管理系统后台项目

## 开发环境

- Node.js >= 8
- npm >= 5
- mongoDB >= 3.6

## 启动项目

``` bash
# install dependencies
npm install

# mail config
cp mail.config.example.js mail.config.js

# serve with hot reload at localhost:4000
npm start

# build for deploy
npm run build
```
## 技术栈
- Node.js
- Express
- Webpack
- ES6
